# 🖼️ Image Studio - 图像批量处理工作台

> 一款纯前端、无需安装的批量图像处理工具。支持背景填充、比例缩放、智能裁剪、格式转换与批量导出，所有计算均在浏览器本地完成，安全高效。

## ✨ 核心功能

| 功能模块 | 说明 |
|---|---|
| 🌐 **多语言界面** | 内置 中文 / English / 日本語 一键切换，开箱即用 |
| 🎨 **背景填充 & 比例适配** | 预设 1:1, 4:3, 3:4, 16:9, 9:16 等常用比例，支持自定义宽高比与背景色，轻松适配电商、社媒平台规范 |
| 📏 **分辨率缩放** | 支持锁定长宽比缩放，或按“长边最大像素”智能约束，快速统一图片尺寸 |
| 🔄 **格式转换 & 质量控制** | 输出可选：保留原格式 / JPEG / PNG / WebP。JPEG/WebP 支持 `1-100` 质量滑块，自由平衡体积与画质 |
| ✂️ **可视化智能裁剪** | 支持按比例裁剪与可选 Smartcrop 主体检测。提供拖拽式裁剪框，支持边角调整、ESC关闭、← → 键切图，操作直观 |
| 📦 **批量处理 & 导出** | 拖拽多选上传，处理完成后可一键打包为 `.zip` 下载，或逐个保存 |
| 📜 **实时操作日志** | 底部日志面板实时反馈处理状态，批量任务进度一目了然 |

## 🚀 快速开始

### 💻 本地运行
本项目为单文件静态应用，无需任何服务器或构建环境：
1. 下载或克隆本仓库
2. 双击打开 `index.html`即可在浏览器中直接使用
3. 推荐使用 Chrome、Edge、Firefox 等现代浏览器以获得最佳性能

## 🔒 隐私与安全
- **纯本地处理**：所有图片的读取、编辑、转换均通过浏览器原生 `Canvas API` 与 `File API` 在内存中完成。
- **零数据上传**：图片**不会**发送至任何第三方服务器，彻底杜绝隐私泄露风险。
- **无追踪机制**：不收集用户数据、不设置 Cookies、无外部分析脚本。

## 🛠️ 技术栈
- 原生 HTML5 / CSS3 / JavaScript (ES6+)
- Canvas API & Blob API & URL.createObjectURL
- `JSZip`（用于客户端批量打包下载）
- 零框架依赖，单文件架构，轻量且加载迅速

## 📝 使用提示
- 上传后点击缩略图可进入**全屏裁剪模式**，直接拖拽调整区域或移动裁剪框。
- 处理过程中可随时点击「清除缩放」或「取消裁剪」恢复至原始图片状态。
- 若启用 `Smartcrop` 智能检测，请确保浏览器支持相关分析接口（可选功能，不影响基础裁剪）。
